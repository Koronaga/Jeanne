Shinobu

Configuring Shinobu for your Server

Welcome Message

s$settings welcome <channel> <message>

Channel: Either DM or #channel.
Message: The messsage to post.

You can add the following placeholders to the message:

  • ${SERVER} The name of the server.
  • ${USER} The name of the user.
  • ${MENTION} Mention the user.

To disable welcome messages do s$settings welcome disable.

Event Notifications

s$settings events <channel> <events>

Channel: The #channel to post events in.
Events: The events to subscribe (+event) or unsubscribe (-event) from.

These are the avaliable events:

  • memberjoined A user joins the server.
  • memberleft A user left the server.
  • userbanned A user was banned.
  • userunbanned A user was unbanned.
  • namechanged A user changed their name.
  • nicknamechanged A user changed their nickname.

Command Ignoring

s$settings ignore (scopes) (commands)

Scopes: Either a @User, #channel, or the server.
Commands: Either a s!command, cleverbot, s!all of a prefix, or all commands.

Scopes and commands are seperated with a |.
Example: s$settings ignore (@User | @User 2 | #no-bot) (s$all | s!ping | cleverbot)

Unignoring works the same way. Just use unignore instead.

If there is only one scope you can ommit the parenthesis.

NSFW Channels

s$settings nsfw <allow/deny>

Allow or deny NSFW commands in the channel.

Checking settings

s$settings welcome check
s$settings events check
s$settings nsfw check
s$settings ignored <server|@user|#channel>
